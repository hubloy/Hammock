The best WordPress Membership Plugin. Restrict content, create rules, collect payments and more.

== Description ==

Memberships by Hubloy gives you all the tools you need to start, manage, and grow your membership site.
Users have full control of their membership and subscription on your site.

[Documentation](https://hubloymembership.com/docs/)

= Member Experience Features =

* Frontend Log In
* Frontend User Registration
* Frontend Password Recovery
* Frontend Profile Editing
* Membership Account Dashboard
* Membership Invoices and Billing Information Pages
* Custom Emails
* Custom Page Templates

== Installation ==

= Download, Install and Activate! =
1. Go to Plugins > Add New to find and install Memberships by Hubloy.
2. Or, download the latest version of the plugin, then go to Plugins > Add New and click the "Upload Plugin" button to upload your .zip file.
3. Activate the plugin.
